# Epidemic SIR Simulator

Interactive web-based simulation of the SIR epidemiological model, developed to visualize disease spread dynamics in a population.

## Features

- Interactive epidemic simulation
- Susceptible / Infected / Recovered model
- Dynamic visualization
- Adjustable simulation behavior
- Responsive frontend interface

## Technologies

- HTML5
- CSS3
- JavaScript

## Project Structure

```bash
index.html     # Main application
script.js      # Simulation logic
styles.css     # Styling
```

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/epidemic-sir-simulator.git
```

Open the project:

```bash
cd epidemic-sir-simulator
```

Run locally by opening:

```bash
index.html
```

## About the SIR Model

The SIR model is a mathematical model used to simulate how infectious diseases spread through populations.

The model divides the population into:

- Susceptible (S)
- Infected (I)
- Recovered (R)
